# Changelog

All notable changes to the SmoothSend SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of SmoothSend SDK
- Multi-chain support for Avalanche and Aptos
- Gasless transaction execution
- TypeScript support with comprehensive type definitions
- Event system for real-time transaction tracking
- Unified API across all supported chains
- Batch transfer support for Avalanche
- Comprehensive error handling and validation
- HTTP client with retry logic and proper error handling
- Chain-specific adapters with extensible architecture
- Balance and token information queries
- Address and amount validation utilities
- Complete documentation and examples

### Features
- **Avalanche Support**: EIP-712 signatures, permit-based gasless transfers
- **Aptos Support**: Native transaction signing, USDC fee payments
- **Developer Experience**: Type-safe API, event listeners, unified interface
- **Security**: Client-side signing, comprehensive validation, rate limiting
- **Extensibility**: Plugin architecture for adding new chains
